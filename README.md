# TicTacToePlus

A strategic Tic-Tac-Toe variant developed using the **Ring programming language** and **RingQt**. This version features a clean, functional interface built entirely through code.

## New Features
*   **Undo Capability:** A dedicated **"Delete"** button in the top-right corner allows you to undo the last move.
*   **Coordinate Display:** Real-time feedback showing the exact (Column, Row) coordinates of the last action.

## Technical Highlights
*   **Pure Code UI:** The graphical interface is constructed programmatically without using a visual form designer, ensuring full control over the layout.
*   **100% Ring:** Built entirely using the Ring language and its native extensions.

## Requirements
*   [The Ring Programming Language](https://ring-lang.github.io)
*   RingQt extension (included in the standard Ring distribution)

## How to Run
1. Ensure Ring is installed.
2. Download the repository files.
